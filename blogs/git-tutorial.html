Using Git to Update Your Homepage

Step 1: Preliminaries (Windows)

1.1 Download

1) latest notepad++
Note: please choose 32-bit version
2) latest git

1.2 Installation

1) Continue pressing "next" when installing notepad++
2) Cancel "Git GUI Here"
3) Choose "Use Notepad++ as Git's default editor"

Step 2: Create Your Homepage

2.1 Sign up

register a github account
Note: please remember your email & username

2.2 Create Repository

The suffix of repository name is "github.io", for example, my username is "inouetoukyou", so my repository name is "inouetoukyou.github.io".

Step 3: Configure the Git

3.1 Set up Email and Username

1) From a directory where you want to store your homepage files, right click, "Git Bash Here"



2) Type command:
git config --global user.name "杨 明航(Minghang Yang)"
Set username, please use your own name, rather than "杨 明航(Minghang Yang)"
git config --global user.email "inouetoukyou@xxxxxx"
Set email, please use your own email
Note: you just need to config it once.


3.2 Generate a SSH Key

1) Type command:
ssh-keygen.exe
Generate key
Note: it's recommended to set up a password.
2) Add the public key to github :
By default, the file "id_rsa.pub" is in "C:\Users\{Your User Name}\.ssh"
Open it and copy the key.
Paste it to GitHub.

Step 4: Push

4.1 Clone Your Repository

Type command:
git clone git@github.com:inouetoukyou/inouetoukyou.github.io
prototype: git clone git@github.com:{your user name}/{your repository name}
cd inouetoukyou.github.io


4.2 Build Your Homepage

1) Modify index.html
Create a file "index.html", then you type command:
git status
See current status.
git add index.html
If you update many files (add, modify or delete), you can type:
git add - A
git commit -m "init index.html"
prototype: git commit -m "{Your commit}"
Note: {Your commit} is necessary
git push origin master
Note: you can type gitk to see your update logs.



2) Visit the homepage
https://inouetoukyou.github.io
